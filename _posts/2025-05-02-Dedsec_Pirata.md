<h1 align="center">DEDSEC_PIRATA</h1> 
<h4 align="center">Command-line for plundering torrent files - search, discover, and save .torrent files for movies, cracked apps, music, OS ISOs, and more.</h4>

## DESCRIPTION

DEDSEC_PIRATA is a powerful terminal-based torrent hunter built for those who want full control over what they download. Instead of directly using magnet links or clients, PIRATA lets you search from multiple torrent indexers and save .torrent files locally â€” allowing you to use your own torrent client for full privacy and flexibility.
With a minimal interface and fast execution, it's the perfect tool for users who want a CLI-first, no-nonsense approach to collecting torrents without opening a browser.

## FEATURES
  * Torrent Search Engine: Search across multiple torrent indexers.
  * .torrent File Saver: Download and save .torrent files directly to your machine.
  * Multi-category Targeting: Find torrents for Movies, Cracked Software, Music, Games, OS ISOs, and more.
  * Minimal CLI Interface: Built for terminal lovers, no GUI required.
  * Formatted Output: Results displayed cleanly using tabulate.
  * No Browser Needed: Full interaction in the shell.
  * Built for Pirates: A digital cutlass for navigating the seas of shared files.

## INSTALLATION 
    * git clone https://github.com/0xbitx/DEDSEC_PIRATA.git 
    * cd DEDSEC_PIRATA
    * chmod +x dedsec_pirata
    * ./dedsec_pirata 

    or

    * sudo apt install ./dedsec_pirata.deb
    * dedsec_pirata
