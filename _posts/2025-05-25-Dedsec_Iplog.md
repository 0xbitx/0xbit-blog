<h1 align="center">DEDSEC_IPLOG</h1>
<h4 align="center">An advanced IP logging and tracking toolkit with real-time data capture and smart URL masking</h4>

### DESCRIPTION

**IPLOG** is an advanced IP logging toolkit designed to capture detailed visitor information as soon as a target opens a link. Built for cybersecurity professionals, ethical hackers, and researchers, it offers powerful tracking features in a streamlined, user-friendly interface.

When a target accesses a generated link, the tool captures:
- IP Address  
- Device Type  
- Screen Resolution  
- Country Name  
- ISP  
- Latitude  
- Longitude  

### Key Features:
- **Real-Time Data Logging** – Instantly collects and displays information when the link is opened  
- **Advanced URL Masking** – Generates convincing, lookalike links to increase engagement and disguise intent  
- **60+ Custom URL Templates** – Choose from a wide range of templates to suit your use case  
- **User-Friendly Interface** – Clean, modern design with zero manual setup required  
- **Fully Automated Workflow** – Just generate a link, share it, and view the results as they come in
